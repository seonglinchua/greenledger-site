# GreenLedger

GreenLedger is a modern sustainable finance website.

The goal is to build a professional, content-led informational site that explains sustainable finance clearly for investors, institutions, and businesses.

## v1 scope
Build a polished responsive single-page website with these sections:
- Hero
- Why it matters
- Core pillars
- Use cases
- Insights
- Trust / methodology
- Contact / CTA

## Stack
- HTML
- CSS
- Minimal JavaScript

## Files
- `index.html`
- `styles.css`
- `script.js`
- `AGENTS.md`
- `docs/content-outline.md`

## Goals
- Clean finance-style design
- Restrained green accents
- Clear content hierarchy
- Easy deployment to shared hosting or VPS
- No backend for v1
